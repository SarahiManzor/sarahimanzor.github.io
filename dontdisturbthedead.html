Don't Disturb the Dead

This is both my first completed game using Unreal Engine 4 as well as my first game jam. I had been learning the engine as well as blender in the couple months leading up to this jam. I felt that I had learnt enough to create a game of my own that I would be proud of. This game was submitted to the Weekly Game Jam 150 with the theme "You are the enemy" which had a 6 day development period. Everything you see in the game was created by me from scratch by me, except for the sound.

The objective of this game is to dig up specific graves described to you through the surroundings. Every grave dug would also result in a ghost chasing you, whether correct or incorrect.
