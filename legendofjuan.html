Legend Of Juan

About this project

Development period: June 2014

Lines of code: 1278

Estimated time spent on development: ~20 hours

Description: You start out as a character named Juan who is only able walk around and encounter NPCs, which provide you with new abilities that help you progress through the game.

This game was inspired by the classic Legend of Zelda, which is one of my all time favourite games. This was my first real game development project. This project had a huge influence on my decision to pursue game development.
