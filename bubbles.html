Bubble Shooter

About this project

Development period: July 2014

Total lines of code: 1317

Estimated time spent on development: ~15 hours

Description: The goal of this game is to survive as long as you can against waves of enemies. Each enemy has a chance of dropping various powerups which can help you live longer.

I really enjoyed working on this game and it was one of my favourite projects. It was made with the help of Youtube tutorials as it was one of my learning projects. Working on this greatly helped me understand the core structure of the game development process.
