Canon Craze

Development Period: July 2015 - September 2015

My first experience working in Unity was working on this game. I was attempting to create my own mobile version of a game called "Toss the Turtle" which I grew up playing. I was not able to complete it but I was still able to create a working prototype. Working on this project really improved my skills with the Unity engine, as well as improving my general programming and problem solving skills.
