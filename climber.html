Crazy Climber

About this project

Start of Development: April 14th 2017

Completion date: April 30th 2017

Total lines of code: 1042

Estimated time spent on development: ~30 hours including design.

Description: The goal of this game is to climb beanstalks as high as you can while dodging falling obstacles. This game gets gradually harder as time passes, in both easy and hard mode. Easy mode consists of the player moving up the screen at a constant rate, while hard mode requires you to alternate between left and right taps to move up.

The purpose of this project was to help my understanding of the process required to complete and publish a game of my own. This project also helped me learn how to create my own assets and animations.
