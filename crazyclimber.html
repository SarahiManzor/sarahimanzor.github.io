Crazy Climber

Development Period: April 14th 2017 - April 30th 2017

This is the first and only game I have published so far. The goal of it was to help my understanding of the process behind creating and publishing a game from the ground up. Everything in this project was made by me, except for the music. I really enjoyed working on this project as the scope was not too large which allowed me to complete it in a fairly short time frame while only allocating a few hours a day to it.

The objective of the game is simple, you just need to climb as high as you can while avoiding falling beans.
